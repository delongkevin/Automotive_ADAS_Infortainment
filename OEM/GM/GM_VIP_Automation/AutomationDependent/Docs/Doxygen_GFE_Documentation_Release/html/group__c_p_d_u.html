Function that checks if an Autosar PDU cycle time is in a given range [aPDUCycleTimeMS - (aPDUCycleTimeMS * aPermittedTimeDiffPerCent/100), aPDUCycleTimeMS + (aPDUCycleTimeMS * aPermittedTimeDiffPerCent/100)] or not for a given number sample times.

Parameters
[in]aPDUNameFromDB- Name of the Autosar PDU from the arxml database.
[in]aNumberOfSampleTimes- The number of samples time to test that the cycle time of Autosar PDU is in range or not
[in]aPDUCycleTimeMS- The FrPDU cycle time you want to check
[in]aPermittedTimeDiffPerCent- The permitted time difference of the upper and lower limit of the cycle time range (per cent).
[in]aBusContext- The bus context.
[out]aPDUVerifyCyclicTime- Extra information regarding the function output (displayed in the test report).
Returns
0 - The permitted time difference value is wrong, the given cycle time value is wrong, the bus context name is wrong, or the cycle time is not in given range or the number of sample times is wrong .
1 - The cycle time is in given range.
-1 - The Autosar PDU does not been received on the bus or an internal error occurred.

Function that checks if a Autosar PDU is received in a specific time or not, with the expected Equal/NotEqual payload data.

Parameters
[in]aPDUNameFromDB- Name of the Autosar PDU from the arxml database.
[in]aExpectedPayloadData- Byte array with the exact size and desired content of what the received payload should be identical/different (depending on the aExpectedCompareAction).
[in]aExpectedCompareAction- The compare action between the provided array and received payload: Equal or NotEqual.
[in]aWaitingTime- Maximum wait time, in milliseconds, to check if the Autosar PDU arrives or not.
[in]aBusContext- The bus context.
[out]aPDUDataCompareInfo- Extra information regarding the function output (displayed in the test report).
Returns
0 - Unexpected content arrived, Autosar PDU did not arrive at all or the size of the array is not correct.
1 - The payload data which arrived is as expected (based on aExpectedCompareAction: 0-Equal/1-NotEqual).
-1 - Conversion has failed or another error occurred (check the Error Code).
