enum cc_eTimeFormat  aTimeFormat,
enum cc_eTimeCmp  aTimeCmpType 
)

Function that check if the requested time for the last wait instruction is as expected or not.

Parameters
[in]aTimeToCompare- Time to compare.
[in]aTimeFormat- How the time will be compared in [nS, uS, mS, S] -> see definition of enum cc_eTimeFormat.
[in]aTimeCmpType- Compare Type - TimeLessEq, TimeMoreEq (less or equal then last elapsed time, more or equal then last elapsed time)
Note
Use this function when is needed to see if the event is occured after a certain time.
