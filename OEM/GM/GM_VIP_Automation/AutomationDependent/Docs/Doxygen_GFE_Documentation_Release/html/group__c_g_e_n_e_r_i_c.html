Function that will get the last function wait elapsed time in requested format.

Parameters
[in]aTimeFormat- How the time will be returned [nS, uS, mS, S], see definition of enum cc_eTimeFormat.
[out]aMappedResponse- Value in string format or message error.
Returns
>0 - Elasped time for the last wait instruction
-1 - Invalid time format or no wait time for the last instruction
Note
Use this function when is needed to see if the event is occured after a certain time.
