Function is going to parse each file line and search within for the provided Keyword/Phrase.

Parameters
[in]aAbsoluteFilePath- The absolute file path (example: C:\Users\romulus.bita\Desktop\FileToBeOpened.txt).
[in]aKeywordToSearch- The keyword or phrase (480 char limit) which is searched within the file (examples: "My Keyword", "c0:18:85:5f:55:cf", "192.168.7.94").
[out]aSearchKeywordInfo- Extra information regarding the function output (displayed in the test report).
Returns
0 - The keyword/phrase has not been found within the file.
1 - The keyword/phrase is found within the file.
-1 - Empty or too large keyword provided, file could not be opened or another internal error occurred
Note
  • Avoid providing only special characters as these are not considered on their own, as a keyword(ex: "...")
  • Empty keywords are restricted as there is nothing to search in this case(ex: "")
