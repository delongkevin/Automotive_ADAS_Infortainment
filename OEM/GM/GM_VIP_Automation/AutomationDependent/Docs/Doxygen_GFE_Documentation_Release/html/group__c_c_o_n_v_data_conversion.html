byte bFctn_CONV_ByteArrayToSpecificNumberBaseString ( byte  aRawByteArrayData[],
enum cc_eNumberBaseCONV  aNumberBase,
char  aSeparationChar[],
char  aOutputString[],
dword  aNumberOfBytes2Conv 
)

Function that will convert byte array elements to a specific number base string (ex: Binary, Decimal, Hex).

Parameters
[in]aRawByteArrayData- The byte array to convert.
[in]aNumberBase- Number base to be used when conversion is done to string (cc_eCONV_Binary = 2, cc_eCONV_Decimal = 10, cc_eCONV_Hex = 16).
[in]aSeparationChar- A string to add between each byte, may be empty (Empty: "", Whitespace: " ", Hex: "0x").
[in]aOutputString- A character array that upon success will contain the converted values (based on the specified number base).
[in]aNumberOfBytes2Conv- How many elements from the byte array will be converted to string.
Returns
1 - Ok, conversion successfully done.
0 - Not ok, error during the conversion phase.
Note
-Example: byte array 0xAA, 0xBB, 0xCC, 0xDD, 0xEE is converted in: -decimal: 170 187 204 221 238 -binary: 10101010+10111011+11001100+11011101+11101110 -hexa: aa+bb+cc+dd+ee

◆ bFctn_CONV_ConvertStringToFloatingNumber()

byte bFctn_CONV_ConvertStringToFloatingNumber ( char  aInputString[],
double &  aFloatingPointOutValue,
char  aConversionInformation[] 
)

Function that will convert a string to a floating point number.

Parameters
[in]aInputString- The string which will be converted.
[out]aFloatingPointOutValue- The converted value.
[out]aConversionInformation- Extra information regarding the function output.
Returns
1 - The conversion is fine.
0 - An error occurred - check aConversionInformation.
Note
- Data Types that are recommanded to use for this function are the floating point ones: Type_Float and Type_Double.
  • The input string must have the format: [whitespace][sign][digits][.digits][(d|D|e|E)[sign]digits], where d/D/e/E are exponents
  • Example: ~"-1.23" is converted to 1.23 ~"2.4E3" is converted to 2400 (2400 = 2.4 * E^3 = 2.4 * 10^3)
