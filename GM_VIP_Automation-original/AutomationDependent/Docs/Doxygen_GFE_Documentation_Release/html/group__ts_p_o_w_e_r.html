void A_POW_SetCommand ( enum cc_eCommandToSend  aCommandToSend,
char  aValue[],
byte  bPSNumber 
)

Function sets and send a specific command to the selected Power Supply.

Parameters
[in]aCommandToSend- Commands to send can be: eSetVoltage / eSetCurrent / ePowerSupply_ON / ePowerSupply_OFF
[in]aValue- Value to be sent for setting voltage/current. It must be format of: XX.xxx V/A. For ON or OFF, it can be empty
[in]bPSNumber- power supply selected (0,1,...7 - from ini file).

Function sets and send a specific command to the selected Power Supply, in order to get a response from it.

Parameters
[in]aCommandToSend- Commands to send can be: eGetStatus, eGetVoltage, eGetCurrent
[in]aValueToCompare- Value to compare the status, voltage or current. Format: XX.xx V/A or 1/0 for Status
[in]aTolerance- Tolerance for comparing.
[in]aPSNumber- power supply selected (0,1,...7 - from ini file).
