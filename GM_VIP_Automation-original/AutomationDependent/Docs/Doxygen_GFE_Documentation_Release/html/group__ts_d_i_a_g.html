◆ A_DIAG_SendAndGetPrimitiveData() [2/2]

void A_DIAG_SendAndGetPrimitiveData ( diagRequest *  aDiagReq,
dword  aStartByte,
dword  aStopByte,
byte  aPrimitiveDataOuputArray[] 
)

Function that sends a specific request and stores the primitive data, between start and stop byte position, in an output byte array.

Parameters
[in]aDiagReq- Diagnostic request.
[in]aStartByte- The byte position from where the data will START to be copied (check note).
[in]aStopByte- The byte position where the data will STOP copying.
[out]aPrimitiveDataOuputArray- The output array where the primitive data will be copied.
Note
Based on how the request is defined, if a specific target is necessary, first use the function A_DIAG_SetTarget(). 
Counting starts from the "Data" response, without taking into consideration the eco bytes from service and DID.
  Example: 
  Request:          22 D4 00
  
Positive respose: 62 D4 00 01 00 23 01 9B 00 78 01 66 8C 1E
                  |   \/   \_____________________________/
                  serv DID                Data
  Counting starts from byte number 0 (elements on the start and end position are copied as well).
  "Data" response byte 0 is 0x62,
  "Data" response byte 1 is 0xD4,
  "Data" response byte 2 is 0x00
  "Data" response byte 3 is 0x01, etc...
  
  If start byte = 2, stop byte = 5 => selected bytes: 0001 00 23
  If start byte = 0, stop byte = 0 => selected byte is: 0x62
