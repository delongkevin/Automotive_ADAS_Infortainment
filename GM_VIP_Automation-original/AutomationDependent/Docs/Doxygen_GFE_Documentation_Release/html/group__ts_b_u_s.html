Function Wait and check if a signal is send with a specific pysycal content (the content of signal cannot be represented on more than 32 bits).

Parameters
[in]aSignalName- Signal name from the database.
[in]aTimeToWaitForSignal- Maximum time to be waited (if the value is 0 the function should get the first value of the signal)
[in]aSignalValue- The value that will be compared with the value of the signal.
[in]aExpectedResult- The compared value is expected to be equal or not equal than signal value.
Returns
no return
Note
  • An timeout must be set before this step (when aTimeToWaitForSignal is zero )
