Function sets the parameters for communication on RS232.

Parameters
[in]bPort- port number.
[in]iBaudRate- baud rate for selected device.
[in]bDataBits- number of data bits.
[in]bParity- parity.
[in]bStopBits- number of stop bits.
[in]bType- type of POW.
[in]bWaitTime- wait time for answers.
[in]lMaxVoltage- Maximum voltage allowed.
[in]lMaxCurrent- Maximum current allower.
[in]iSCPIAddress- The address of the SCPI power supply. (It is used only for SCPI PS)
[in]iFlowControl- The FlowControl of the power supply. ( 0 - no FlowControl at all, 1 - hardware FlowControl, use DTR<->DSR )
[in]lWaitForCommand- The waiting time between seding commands
