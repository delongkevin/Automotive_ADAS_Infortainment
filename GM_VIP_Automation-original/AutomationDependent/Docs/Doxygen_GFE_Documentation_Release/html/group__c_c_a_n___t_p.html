char  aNodeClusterName[],
byte  aArrayToStoreData[],
dword  aWaitingTime,
char  aGetDataInfo[] 
)

Function that waits an amount of time, on a certain node and copy the first arrived data in the provided output byte array.

Parameters
[in]aNodeName- Name of the node where the data is expected to be received and saved from.
[in]aNodeClusterName- Name of the Cluster on which the node is present.
[out]aArrayToStoreData- Output byte array with first received data from the node (should be large enough in order to save the complete data).
[in]aWaitingTime- Maximum wait time, in miliseconds, to wait for the data to arrive and save it (cannot be < 1ms).
[out]aGetDataInfo- Extra information regarding the function output (displayed in the test report).
Returns
0 - Data did not arrive on the specified node and within the waiting time, or incorrect node name was provided.
1 - Data has arrived on the specified node within the wait time and has been successfully copied in the output array.
-1 - Node is not enabled, conversion has failed or another error occurred.
